# Banking Aspirant Manager Bot

## Features

- Welcome Screen
- Subject-wise Notes
- Google Drive Integration
- About Page
- SQLite Database
- Admin Stats
- Broadcast Messages

## Installation

```bash
git clone <repository>
cd BankingAspirantBot

py -m venv venv

venv\Scripts\activate

pip install -r requirements.txt
```

Create a `.env` file:

```
BOT_TOKEN=YOUR_BOT_TOKEN
ADMIN_ID=YOUR_ID
```

Run:

```bash
python bot.py
```
